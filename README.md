# ATM Termos y Mates

ATM Termos y Mates es una aplicación web desarrollada para facilitar la compra de termos, mates y vasos, brindando una experiencia de navegación rápida, intuitiva y organizada.

La aplicación permite a los usuarios explorar un catálogo de productos donde cada artículo muestra su imagen, nombre, precio y un descuento del 15% aplicado automáticamente. Además, incorpora un buscador para encontrar productos de forma rápida y sencilla.

Entre sus principales funcionalidades se encuentra un carrito de compras que permite agregar productos, modificar cantidades, eliminar artículos y visualizar el total de la compra antes de finalizar el pedido.

También cuenta con un sistema de inicio de sesión que permite identificar a los usuarios y sienta las bases para futuras funcionalidades, como la gestión de pedidos, el almacenamiento de información del cliente y la incorporación de métodos de pago online, con el objetivo de completar todo el proceso de compra dentro de la aplicación.
